## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9
- MongoDB
- Redis

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/temp-mail.git
cd temp-mail
```

2. Install dependencies
```bash
npm install
```

3. Start development servers
```bash
npm run dev
```
